structured docstrings
=====================

Try ::

   pydoc structdoc_example

or ::

   >>> import structdoc_example
   >>> print(structdoc_example.func.__doc__)
   >>> structdoc_example.otherfunc.__doc__

or ::

   In [1]: import structdoc_example

   In [2]: structdoc_example.func?
